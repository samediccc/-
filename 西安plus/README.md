# 西安时空地图 - 前端课程设计报告

---

## 1. Project Introduction

本项目是一个基于 Web 技术的交互式地图应用，旨在通过可视化的方式展示古都长安与现代西安的地理变迁与历史风貌。用户可以通过时间轴切换、地标交互和数据故事等功能，探索西安这座城市从古至今的发展脉络，感受其深厚的历史文化底蕴。

## 2. Tech Stack

本项目为纯前端应用，不依赖任何后端服务。主要技术栈如下：

-   **核心语言**: HTML5, CSS3, JavaScript (ES6)
-   **开发工具**: Vite
-   **数据可视化**: SVG (用于绘制地图、图标及基本图表)
-   **代码规范**: Prettier (推荐)
-   **版本控制**: Git

## 3. Getting Started

请确保你的开发环境已安装 [Node.js](https://nodejs.org/) (推荐 v16.x 或更高版本)。

1.  **克隆仓库**

    ```bash
    git clone <your-repository-url>
    cd <project-directory>
    ```

2.  **安装依赖**

    在项目根目录下打开终端，运行以下命令安装项目所需依赖：

    ```bash
    npm install
    ```

3.  **启动项目**

    安装完成后，运行以下命令启动本地开发服务器：

    ```bash
    npm run dev
    ```

    服务启动后，终端会显示访问地址 (通常是 `http://localhost:5173`)。在浏览器中打开该地址即可查看项目。
